
# Teachers - CRUD

Teachers-CRUD (really imaginative, I know) is a simple yet fully-fledged CRUD app made to store teachers' data for the AUEB institution.





## Tech Stack

**Client:**  TailwindCSS, daisyUI

**Server:** Spring (Boot, Data JPA), PostgreSQL
